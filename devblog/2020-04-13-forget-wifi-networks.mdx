### Summary

This `2020.04.13.0` release upgrades gateway-config to enable the removal of saved Wi-Fi networks
from hotspots via the mobile app. A bug fix for a recent regression that disabled GPS on the
hotspots is also included.

<!--truncate-->

### Contents

- _Add configured Wi-Fi services characteristic_: PR:
  [gateway-config/#57](https://github.com/helium/gateway-config/pull/57)
- _Add GATT characteristic for removal of Wi-Fi networks_: PR:
  [gateway-config/#56](https://github.com/helium/gateway-config/pull/56)

### Plan

A `2020.04.13.0` release was deployed to the beta group around 1:00 PM PST on Monday April 13th. If
overnight testing goes well we will release `2020.04.13.0` to general availability the following
morning.
