This `2020.07.24.0` release completes the "Forget Network" feature that will be added to an upcoming
version of the Helium mobile app. We also fixed a couple bugs and added debugging to prepare for
upcoming state channels V2 chain variable.

<!--truncate-->

This release also updates miner to handle payerless assert location transactions and bumps the
onboard blessed snapshot to block height `427681`.

### Plan

A `2020.07.24.0` release was deployed to the beta group on the morning of Friday July 24 at around
