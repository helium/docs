# Bug Fixes

This release fixes the websockets used on the main Accounts page (it was using the old API, now uses
the new one) and updates the polling interval to 30 seconds so it can fetch recently cleared pending
transactions.

It also fixes a bug where NaN is displayed for pending transactions.
