The `2020.09.08.0` release is a minor bugfix release to fix an issue with the stabilization interval
calculations on newer Hotspots that have been loaded from a snapshot. There was an error in the how
the interval was calculated, which the PR below fixes:

<!--truncate-->

[Fix Time Targeting on Newer Nodes](https://github.com/helium/miner/pull/505)

Additionally, this release bumps the blessed snapshot to height 487441.

### Plan

We have been betaing `2020.09.08.0` since the afternoon on the 8th and will be release it to GA by
10AM PST.
