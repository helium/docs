The team has a new release tagged `2021.03.02.2`. This release collects several performance and
maintenance upgrades to the blockchain that should improve stability and performance of members of
the Consensus Group. We also included a few code changes that will support Light Hotspots in the
future, a bugfix for oracle prices our testnet chain, and snapshot bump.

We have been beta testing `2021.03.02.2` since March 2nd, 2021 1:00 PM PT and have been testing
prior versions since 11 AM PT today. Current ETA for GA is 1:45 PM PST, March 2nd, 2021.
