# New Features

## Fleet Mode

- Fleet Mode is a new feature for accounts with more than 20+ Hotspots that reduces the API calls it
  makes and improves app performance
- Read more on [Helium Docs](https://docs.helium.com/wallets/app-wallet/fleet-mode/)

- Updates Nebra Onboarding Information
