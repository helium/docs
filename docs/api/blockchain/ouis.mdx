# OUIs

Organization Unique Identifiers

OUIs are used to assign part of the global address space to a given set of
router addresses. Hotspots that see device traffic will look up the router
address for the given device OUI and offer the device data to registered router.

---

## List OUIs

```
GET https://api.helium.io/v1/ouis
```

Lists all OUI transactions known to the blockchain.

This route is `cursor` paginated.

## Get last assigned OUI

```
GET https://api.helium.io/v1/ouis/last
```

Returns the last assigned OUI transaction. The last assigned OUI can be used to
request a new OUI
