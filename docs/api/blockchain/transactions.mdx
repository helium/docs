## Transaction for Hash

```
GET https://api.helium.io/v1/transactions/:hash
```

Fetch the transaction for a given hash.

_Path Parameters_

| param           | Type     | Note                      |
| --------------- | -------- | ------------------------- |
| hash (required) | _string_ | Transaction hash to fetch |
