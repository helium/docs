# Validators

---

:::warning 

These docs reflect the Validator Mainnet API. However, it is **not yet live**. We are currently on Testnet. During the Testnet period, please use the following as the base url for the Testnet API:

```
https://testnet-api.helium.wtf/v1/
```

:::


## List Validators

```
GET https://api.helium.io/v1/validators
```

List known validators as registered on the blockchain.The results are paged. If a
`cursor` field is present in the response, more results are available.

