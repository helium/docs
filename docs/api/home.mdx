# Helium APIs

The Helium Network currently exposes the following APIs:

* [Blockchain API](/api/blockchain/introduction) - Access real-time and historical data from the helium blockchain. 
* [Console API](/api/console) - Provides programatic access to your organization's sensor data on [console.helium.com](https://console.helium.com)
