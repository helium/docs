The Helium Network currently exposes the following APIs:

- [Console API](/api/console) - Provides programatic access to your organization's sensor data on
  Helium console instances.

All other data can be accessed through [Oracles](/oracles/).

Note: the free to use Foundation Console [console.helium.com](https://console.helium.com) does not
support the Console API.

Console providors that do support the Console API are found at
[Console Hosting Providers](/use-the-network/console/hosting-providers)
