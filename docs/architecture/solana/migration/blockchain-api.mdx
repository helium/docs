The legacy blockchain API was retired on July 28, 2023.

To accommodate the need for historical data, a CSV export of the Helium L1 data is available on a
requestor-pays basis on AWS in the `foundation-helium-l1-archive-requester-pays` S3 bucket.

The data is also available as a torrent.

:::caution Notice: January 2024

It has been flagged that these sources miss some early blocks, starting at block ~73k. A new data
source has been retained and a new bucket and torrent file will be generated to provide a complete
history of the Helium L1. The existing sources remain available for those not needing these early
blocks.

Those seeding the above torrent to the community are encouraged to replace it with the updated
version.

This page will be updated as these updated sources become available.

:::

For info on the historical DeWi ETL query tool, check the

For ongoing access, the [Oracle Data](/oracles/oracle-data) and Solana blockchain represent Helium
state after the migration.
