Locations are used to associate human readable geographic informaion to an h3 location. This API
exposes this geographic information.

---

## Deprecated Endpoints

### Get Location

:::caution This Endpoint Is Deprecated

This endpoint is deprecated.

:::

```
GET https://api.helium.io/v1/locations/:location
```
