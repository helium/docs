Nova Labs (formerly Helium Systems, Inc.) manufactured the first set of Hotspots on the Network.
Nova Labs no longer manufacturers Hotspots, but there is now an extensive
[list of manufacturers who sell approved Hotspots](/hotspot-makers).

:::tip Wallet Functionality

Use the [Helium Wallet app](/wallets/helium-wallet-app) to interact with tokens, take part in
governance, and all other non Hotspot management functions.

:::

## Helium Hotspot App Overview

The Helium Hotspot apps for [iOS](https://apps.apple.com/us/app/helium-hotspot/id1450463605) and
[Android](https://play.google.com/store/apps/details?id=com.helium.wallet) include everything needed
to on-board and maintain an Original Helium Hotspot or FreedomFi Hotspot.

The app can be used to:

- Add Nova Labs or FreedomFi Hotspot to the Helium Network
- Set or Modify the Hospot's location
- Configure the Wi-Fi connection of the Hotspot

## Original Helium Hotspot Warranty

The Helium Hotspot limited warranty is only valid for the original sale to the original owner.
Resold Helium Hotspots from anyone other than Helium directly are not under any warranty whatsoever.
Please refer to their documentation for any authorized reseller policy.
