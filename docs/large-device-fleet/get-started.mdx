# How to get started

import useBaseUrl from "@docusaurus/useBaseUrl";

* Buy one or more LoRaWAN device
* Get a Helium hotspot (Optional - only if no [Helium coverage](https://explorer.helium.com/) in your area) 
* Add your device(s) to the Router - you get free credit for 10,000 packets
* Send data to your application/dashboard
