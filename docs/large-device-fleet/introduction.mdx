# Introduction

import useBaseUrl from "@docusaurus/useBaseUrl";

This section is for organisations that plan to run large numbers of devices on the Helium network. It is easy to onboard 
millions of devices onto the Helium network without having to learn the details of how Helium and LoRaWAN work.

Some organisations have additional needs and requirement when it comes to IoT and data. The following content describes 
the various components of the Helium network and the economics of operating a fleet of devices on the Helium network. It 
is up to the organisation to choose whether to leverage Helium's Router infrastructure, or choose to run its own while 
benefiting from the coverage provided by Helium hotspots.
