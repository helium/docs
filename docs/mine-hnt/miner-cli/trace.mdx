# Trace Commands
```bash

Usage: blockchain trace commands

  trace start   - Start a trace for a given filter and level.
  trace clear   - Clears all installed traces.
```
## Start

```bash
miner trace start [ -l|--level [none|debug|info|notice|warning|error|critical|alert|emergency] -o|--output <filename>]
```

  Start a console trace. Use key=value arguments to set
  the trace filters.

## Cancel

```bash
miner trace cancel
```
Clears all installed traces.
