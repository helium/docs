## Add Gateway

```bash
miner txn add_gateway owner=<owner> [--payer <payer>] [--fee <fee>] [--staking_fee <staking_fee>]
```

  Creates a signed add gateway transaction required to add a new Gateway to the Helium network.
  Requires an owner address. Optionally takes a payer address if the payer of the cost and fee is
  not the same as the owner. Also optionally takes the transaction and staking fees (in DC) if the
  caller knows these ahead of time.

  Returns a Base64 encoded transaction to be used as an input to either the
  Helium mobile application or the wallet CLI for signing by the owner, and payer if provided, and
  final submission to the blockchain.

## Asert Location

```bash
miner txn assert_location owner=<owner> location=<location> [--payer <payer>] [--nonce <nonce]
```

  Creates a signed location assertion required to declare the location of a Gateway on the Helium network.
  Requires an owner address and location (in lat,lon or h3 form).
  Optionally takes a payer address if the payer of the cost and fee is not the same as the owner.
  Optionally takes a nonce.

  Returns a Base64 encoded transaction to be used as an input to either the
  Helium mobile application or the wallet CLI for signing by the owner, and payer if provided, and
  final submission to the blockchain.
