# Test cases
Perform the following test cases to help with testing:

* Set up a wallet
* Set up a validator node
* Stake test tokens (TNT) 
* Observe TNT wallet receives rewards
* Unstake TNT, observe:
    * no longer in consensus group
    * no longer receiving rewards
* Cancel unstaking
* Take validator offline
    * prior to consensus group
    * during consensus group
* Restake owner wallet
* Transfer ownership of validator nodes
* Set up wallet to multiple validator nodes

Use the ```#validator``` channel on our [Discord](https://discord.gg/helium) server to provide feedback and report issues.
