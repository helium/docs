# Obtain test tokens (TNT)
After your owner wallet is set up you need to obtain test tokens used to stake the validator node.

To obtain these TNT you need your public key. See View wallet balance and public key section for instructions on how to obtain. 

Go [here](https://faucet.helium.wtf/) and enter your owner wallet public key where specified and follow the onscreen instructions.
