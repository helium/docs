# Obtain Testnet Network Tokens (TNT)
After your owner wallet is set up you need to obtain TNT used to stake the validator node.

To obtain these TNT you first need your wallet public key. See [Configure a Wallet](/mine-hnt/validators/validator-wallet) for instructions on how to get your public key. 

Go [here](https://faucet.helium.wtf/) and enter your owner wallet public key where specified and follow the onscreen instructions.
