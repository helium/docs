Validators are a new entity on the Helium Blockchain that perform the work of the consensus group including verifying transactions and adding new blocks to the blockchain.

## Mainnet

* [Mainnet Deployment Guide](/mine-hnt/validators/mainnet/deployment-guide) - How to get up and running with your a Validator on Mainnet. This guide covers staking, running, updating, and monitoring.
* [Validator Mainnet Explorer](https://explorer-beta.helium.com/validators) - The easiest way to monitor the Validator magic on Mainnet. 

:::warning

**Testnet to Mainnet Migration**

There is no direct migration path to move Testnet Validators to Mainnet. [Details on how to migrate are here.](/mine-hnt/validators/testnet-to-mainnet-migration)

:::
