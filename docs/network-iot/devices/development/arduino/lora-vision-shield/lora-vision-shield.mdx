## Portenta Vision Shield LoRa®

A rapid solution for embedded ML combining vision, audio and connectivity - Detect Gestures,
Identify Objects, See Things. The Vision shield features a 324x324 pixels camera module containing
an Ultra Low Power Image Sensor designed for Always-on vision devices and applications.
