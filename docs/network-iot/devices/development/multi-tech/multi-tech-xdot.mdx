## MultiTech xDot® Long Range LoRa Module

MultiTech xDot®is a secure, end-certified, Arm®Mbed™programmable, low-power RF module, that provides
long-range, low bit rate M2M data connectivity to sensors, industrial equipment and remote
appliances.
