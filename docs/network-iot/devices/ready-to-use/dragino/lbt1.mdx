## LBT1

LBT1 LoRaWAN BLE Indoor Tracker. The Dragino LBT1 is a long-range, low-power consumption LoRaWAN
Bluetooth tracker. LBT1 scans and finds the nearest iBeacon info and sends an LNS server via a
LoRaWAN Network.
