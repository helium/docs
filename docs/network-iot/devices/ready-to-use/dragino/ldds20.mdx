## LDDS20

LDDS20 LoRaWAN Liquid Level Sensor. The Dragino LDDS20 is a LoRaWAN ultrasonic liquid-level sensor
for IoT solutions. It uses ultrasonic technology to measure the liquid level in a container without
opening it and send the value via a LoRaWAN Network to an LNS Server.
