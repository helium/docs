## LDDS75

LDDS75 LoRaWAN Distance Detection Sensor. The Dragino LDDS75 is a LoRaWAN distance detection sensor
for IoT solutions. It measures the distance between the sensor and a flat object. This sensor module
uses ultrasonic sensing technology for distance measurement, and temperature compensation is
performed internally to improve data reliability.
