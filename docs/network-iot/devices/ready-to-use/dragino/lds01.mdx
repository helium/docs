## LDS01

LDS01 -- LoRaWAN Door Sensor

The Dragino LDS01 is a LoRaWAN Door Sensor. It detects the door open/close status and uplinks these
statistics to an LNS server via a LoRaWAN Network. The LDS01 will send periodic data every day as
well as for each door open/close action. It also counts the door open times and calculates the last
door open duration.
