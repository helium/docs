## LGT-92

LoRaWAN GPS Tracker with 9-axis accelerometer LGT-92. The Dragino LoRaWAN GPS Tracker LGT-92 is an
open-source GPS tracker based on ultra-low power STM32L072 MCU and SX1276/1278 LoRa Module. LGT-92
includes a low-power GPS module L76-L and a 9-axis accelerometer for motion and attitude detection.
