## LSN50-V2

LSN50-V2 -- Waterproof Long Range Wireless LoRa Sensor Node. The Dragino LSN50-V2 is a Long Range
LoRaWAN Sensor Node. It is designed for outdoor data logging and powered by a Li-SOCl2 battery for
long-term use and secure data transmission.
