## LSN50v2-D20

LSN50v2-D20 -- LoRaWAN Waterproof / Outdoor Temperature Sensor. The Dragino LSN50v2-D20 is a LoRaWAN
Temperature Sensor for IoT solutions. It measures the temperature of air, liquid or object, and then
periodically reads and uploads data via a LoRaWAN Network to an IoT server.
