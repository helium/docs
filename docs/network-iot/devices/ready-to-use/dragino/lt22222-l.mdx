## LT-22222-L

LT-22222-L LoRaWAN I/O controller. The Dragino LT series I/O Module is a long-range LoRaWAN I/O
Controller. It contains different I/O Interfaces such as: analog current input, analog voltage
input, relay output, digital input and digital output etc. The LT I/O Modules are designed to
simplify the installation of I/O monitoring.
