## izi-io 4840

Powerful Multi-I/O Edge Computing Controller with LoRaWAN. The top-hat rail-mounted device features
8 analog / RTD / digital inputs, 4 digital outputs as well as CAN interfaces for industry-standard
bus protocols such as Modbus. Built-in Soft-PLC can process data on the device for standalone contro
purposes and efficient LoRaWAN usage.
