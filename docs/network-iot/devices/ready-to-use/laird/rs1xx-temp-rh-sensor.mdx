## Sentrius RS1xx Integrated Temperature & Humidity Sensor

The Sentrius RS1xx Integrated Temperature and Humidity sensors enable battery-powered, local and
wide-area sensor applications using LoRaWAN and Bluetooth 4.2! The RS1xx are small, rugged, and
easily configurable sensors making it easier than ever to monitor environmental data with your own
network of sensors.
