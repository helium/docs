## Wireless Water leak Sensor

Netvox R311W is a water leak sensor based on LoRaWANâ„¢ protocol (ClassA) carrying leak detection and
alarm function. When R311W detects a leak, it will send an alarm message to the gateway. When the
sensor detects no leak, it will send a message that reports the status to the gateway. There are two
water leak sensors which allow users to monitor two spots.
