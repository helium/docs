## Wireless Push Button Interface

The device is connected to an external push button device (2 lines are connected to the 2 end of the
push button) that can detect the signal when the button is pushed. It uses SX1276 wireless
communication module, and the communication is fully compatible with LoRaWANâ„¢ protocol (Class A).
