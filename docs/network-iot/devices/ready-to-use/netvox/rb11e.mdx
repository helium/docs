## Wireless Occupancy/Temperature/Light Sensor

RB11E is a Wireless Occupancy/Light/Temperature Sensor for Netvox ClassA type devices based on the
LoRaWAN open protocol and is compatible with the LoRaWAN protocol. It combines occupancy
,temperature, tamper switch and light sensors that suitable for detecting human movement indoors,
illuminance and temperature.
