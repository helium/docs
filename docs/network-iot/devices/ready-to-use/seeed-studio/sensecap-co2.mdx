## SenseCAP Wireless CO2 Sensor - LoRaWAN®

It measures levels of CO2 emission in the atmosphere, designed with an NDIR CO2 sensor and a custom
battery. It's dustproof, optimal for use cases that need reliable data collection, even in dusted
indoor environments such as chicken farms.
