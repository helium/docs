## SenseCAP S2101- LoRaWAN® Air Temperature and Humidity Sensor

SenseCAP 2101 temperature & humidity sensors satisfy industrial long-distance data acquisition with
a wide range of -40°C to 85°C and 0 to 100 %RH respectively. With Bluetooth 5.0 for easy
configuration and firmware upgrade and a built-in replaceable battery for minimal maintenance, it
also supports three different LoRaWAN® network architectures enabling you to easily join it in
existing gateways and network servers to build flexible and high-performance sensor network.
