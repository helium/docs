How to provision a custom-built device on Helium and see the payloads:

- background on LoRaWAN ( keys, onboarding, etc)
- Using an LNS to provision a device
- Create a Device
- Using an off-the-shelf development board.
  - distinction for ready-to-use devices with pre-provisioned keys
- Send data to the Helium Network
- View the data in the Console

With the payloads received, the next step would be to send them to an application. LNSs typically support a number of integrations like HTTP, MQTT, or bespoke integrations with platforms like AWS IoT or Azure IoT.
