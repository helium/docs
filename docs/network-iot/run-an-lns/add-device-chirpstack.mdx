To add a device to your ChirpStack console is currently a 3 step process. Each device needs a device profile and each
device needs to belong to an application. Integrations are currently handled by a per application level for device(s).


 1. Create Device Profile or Select from Template
 2. Create An Application
 3. Create Device in Application


 ## Create Device Profile
 ### Device Profiles -> Add Device Profile

 From the dashboard click on Device Profiles<br />
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_001_dashboard.jpg')} />
 <br />
 <br />

 From Device Profiles click on Add device profile
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_002_device_profile.jpg')} />
 <br />
 <br />

 From here you can either create a new Device Template, or select one from the device-profile templates if they're available.
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_003_device_profile.jpg')} />
 <br />
 <br />

 Click submit to add your new device profile
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_004_device_profile.jpg')} />
 <br />
 <br />

 ## Create Application
 ### Applications -> Add Application
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_005_application.jpg')} />
 <br />
 <br />

 Create application name and description
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_006_application.jpg')} />
 <br />
 <br />


 ## Create Device in Application
 ### Application -> Application-Name -> Add Device
 <br />

 Select your Application and click Add Device
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_007_add_device.jpg')} />
 <br />
 <br />

 Fill in your device details and select the device profile for the device you made earlier.
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_008_add_device.jpg')} />
 <br />
 <br />

 Click submit when done
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_009_add_device.jpg')} />
 <br />
 <br />

 Enter application key
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_010_add_device.jpg')} />
 <br />
 <br />

 Click submit once you enter application key
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_011_add_device.jpg')} />
 <br />
 <br />

 Now your device should be created and listed in the application.
 <img className="docsheader" src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_012_add_device.jpg')} />
 <br />
 <br />
