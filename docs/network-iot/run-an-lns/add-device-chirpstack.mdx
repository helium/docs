To add a device to your ChirpStack console is currently a 3 step process. Each device needs a device
profile and each device needs to belong to an application. Integrations are currently handled by a
per application level for device(s).

1.  Create Device Profile or Select from Template
2.  Create An Application
3.  Create Device in Application

After adding a device to ChirpStack, take note of the Device EUI and Join EUI values in order to be
able to
[Configure Device in Helium Config Service](/iot/run-an-lns/configure-routing-rules#configure-device-in-helium-config-service)
