Adding a device to your ChirpStack console for the first time is currently a 3 step process. Each
device needs a device profile and each device needs to belong to an application. Integrations are
currently handled per-application level for device(s).

1.  Create a Device Profile or Select from Templates
2.  Create an Application
3.  Create a Device in the Application

After adding a device to ChirpStack, take note of the Device EUI and Join EUI values in order to be
able to
[Configure Device in Helium Config Service](/iot/run-an-lns/configure-routing-rules#configure-device-in-helium-config-service)

## Create Device in Application {#add-device}

### Application -> Application-Name -> Add Device

<br />

Select your Application and click **Add Device**.

<figure className="screensnippet-wrapper">
  <img
    className="add-shadow add-shadow-margin add-border-radius"
    style={{ width: '100%', maxHeight: 'initial' }}
    src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_007_add_device.jpg')}
  />
</figure>
<br />
<br />

Fill in your device details and select the device profile you made earlier. A Device EUI can be
entered from a pre-provisioned device, or pressing the **↻** button will generate a fresh EUI.  
The AppKey will be shown in the next step, don't worry about finding it yet.

<figure className="screensnippet-wrapper">
  <img
    className="add-shadow add-shadow-margin add-border-radius"
    style={{ width: '100%', maxHeight: 'initial' }}
    src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_008_add_device.jpg')}
  />
</figure>
<br />
<br />

Click **Submit** when done.

<figure className="screensnippet-wrapper">
  <img
    className="add-shadow add-shadow-margin add-border-radius"
    style={{ width: '100%', maxHeight: 'initial' }}
    src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_009_add_device.jpg')}
  />
</figure>
<br />
<br />

Enter the application key or generate a new one by pressing the **↻** button.

<figure className="screensnippet-wrapper">
  <img
    className="add-shadow add-shadow-margin add-border-radius"
    style={{ width: '100%', maxHeight: 'initial' }}
    src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_010_add_device.jpg')}
  />
</figure>
<br />
<br />

Click **Submit** once you enter the application key.

<figure className="screensnippet-wrapper">
  <img
    className="add-shadow add-shadow-margin add-border-radius"
    style={{ width: '100%', maxHeight: 'initial' }}
    src={useBaseUrl('/img/open-lns/chirpstack/chirpstack_011_add_device.jpg')}
  />
</figure>
<br />
<br />

Your device should be created and listed in the application. The **LoRaWAN frames** tab will display
a live view of payloads being delivered to the LNS.  
Additional devices can be added to the existing Application.
