# Helium Documentation Installation Guide

## Node.js

Download and install [Node.js](https://nodejs.org/en/download) version >= `18.12.1`

```bash
# Verify your Node version with
â¯ node --version
v18.12.1
```

:::info Windows Users

Windows Node.js installation you do not need to install the `Compiler` or `Chocolaty` dependencies.

:::

## Yarn

Download and install [Yarn](https://yarnpkg.com/getting-started/install) version >= `3.3.1`

:::info You Must Set Yarn Version In Each Project Directory

Starting with Node v16.10, the preferred way to manage Yarn is through
[Corepack](https://nodejs.org/dist/latest/docs/api/corepack.html).

```bash
> cd ~/path/to/project
> corepack enable
> corepack prepare yarn@stable --activate
> yarn set version stable
```

:::

```bash
# Verify your yarn version with
â¯ yarn --version
3.3.1
```

### Developing

Each time you save files under your local repo the site will get rebuilt and the pages refreshed on
the fly, there is no need to keep restarting Yarn.
