:::info Using the Helium Solana Ledger App

More information on installing and using the Solana-compatible Helium app for Ledger Wallets in the
[Github Repo](https://github.com/helium/app-helium-solana).

:::

The following utility will migrate all tokens from the Helium derivation path `44'/904'/n'/0'` to
the Solana derivation path `44'/501'/n'/0'`. To get started, install the:

- [Helium Solana Ledger App](https://github.com/helium/app-helium-solana) from Github
- [Solana Ledger App](https://support.ledger.com/hc/en-us/articles/360016265659-Solana-SOL-?support=true)
  using Ledger Live
