The Solana ledger wallet application exists at derivation path 
44'/501'/n'/0'. Helium wallets exist at derivation path 44'/904'/n'/0'.

The following utility will migrate all tokens from the helium derivation path
to the solana derivation path. To get started, install the:

  * Helium Solana Ledger App
  * Solana Ledger App
