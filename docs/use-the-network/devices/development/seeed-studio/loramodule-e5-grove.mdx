## LoRa-E5 Grove

Grove LoRa-E5 embedded with LoRa-E5 STM32WLE5JC, powered by ARM Cortex M4
ultra-low-power MCU core and LoRa SX126x, is an easy-to-use wireless radio
module, supporting LoRaWAN protocol on the EU868 & US915 frequency and (G)FSK,
BPSK, (G)MSK, LoRa modulations.
