# Helium Devices

The Helium network supports any LoRaWAN capable device meeting the
[v1.0.2 specification](https://lora-alliance.org/resource_hub/lorawan-specification-v1-0-2).

:::info

There are thousands of compatible LoRaWAN devices. The devices shown here do not
represent all available devices.

:::

### Complete Device Solutions

Please visit Users under our

offering complete end-to-end device solutions.
