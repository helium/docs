## izi-io 4840

Powerful Multi-I/O Edge Computing Controller with LoRaWAN. The top-hat rail mounted device features
8 analog / RTD / digital inputs, 4 digital outputs as well as industry-standard bus interfaces such
as CAN and Modbus. Built-in Soft PLC can process data on the device for standalone control purposes
and efficient LoRaWAN usage. External power supply with 8..30V.
