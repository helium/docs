## Milesight EM300-TH

EM300-TH Temperature & Humidity Sensor

The EM300-TH is a compact environment monitoring sensor for measuring temperature and humidity. It
transmits data by using LoRaWAN technology. The EM300-TH can work for up to 5 years with 4000 mAh
battery. The EM300-TH is widely used for both indoor and outdoor applications, such as offices,
hospitals, factories, greenhouses, etc.
