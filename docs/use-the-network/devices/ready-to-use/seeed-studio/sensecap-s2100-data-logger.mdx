## SenseCAP S2100 LoRaWAN Data Logger

SenseCAP S2100 LoRaWAN Data Logger is an easy-to-deploy, industrial-grade sensor hub that can
support 100+ industrial sensors and 100+ grove sensors including various sensor types such as
Analog/I2C/UART/RS485.
