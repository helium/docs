# Internal QR Scanner

### Burn HNT

Generated from console

```json
{
  "type": "dc_burn",
  "address": ":address",
  "amount": "1.00",
  "memo": "n0XU06JdNNs="
}
```

### Payment

Single Payee without memo

```json
{
  "type": "payment",
  "address": ":address",
  "amount": 1
}
```

Single Payee with memo

```json
{
  "type": "payment",
  "address": ":address",
  "amount": 1,
  "memo": "n0XU06JdNNs="
}
```

Multiple Payee without memo

```json
{
	"type": "payment",
	"payees": {
		":address1": 10.2,
		":address2": 1.2
}
```

Multiple Payee with memo

```json
{
  "type": "payment",
  "payees": {
    ":address1": {
      "amount": 10.2,
      "memo": "n0XU06JdNNs="
    },
    ":address2": {
      "amount": 1.2,
      "memo": "n0XU06JdNNs="
    }
  }
}
```
